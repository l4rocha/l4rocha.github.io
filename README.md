## GitHub Pages

To publish this site:

1. Create a GitHub repository named `l4rocha.github.io`.
2. Add `index.html`, `style.css` and `README.md` to the repository.
3. Push the files to the `main` branch.
4. In GitHub, open `Settings > Pages`.
5. Set the source to `Deploy from a branch`, select `main` and choose the root folder.
6. Save the settings.

The expected final URL is:

https://l4rocha.github.io
